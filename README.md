# Professional Card Selector

A clean, professional card selection tool built with Streamlit.

## Features
- **Clean UI**: Professional dark theme with smooth animations.
- **Easy Deck Management**: Paste your card list or upload a file.
- **Random Selection**: Draw cards with a bouncy 3D animation.
- **Discard Management**: Keep track of drawn cards.

## Local Setup
1. Clone this repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:
   ```bash
   streamlit run app.py
   ```

## Deployment
This app can be easily deployed to [Streamlit Cloud](https://streamlit.io/cloud).
1. Push this code to a GitHub repository.
2. Connect the repository to Streamlit Cloud.
3. Select `app.py` as the main file.
